Lessons 1–5 (Learn)

Lesson 1 — What is HTML?

HTML structures web pages using tags like <h1>, <p>, and <a>.

<h1>Hello World</h1>
<p>This is a paragraph.</p>

Lesson 2 — Basic Structure

<!DOCTYPE html>
<html>
  <head><title>Title</title></head>
  <body>...content...</body>
</html>

Lesson 3 — Headings & Paragraphs

Use <h1> to <h6> for headings and <p> for normal text.

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<p>This is a paragraph.</p>

Lesson 4 — Images & Links

<img src="https://via.placeholder.com/150" alt="sample">
<a href="https://example.com" target="_blank">Visit</a>

Lesson 5 — Lists & Tables

<ul>
  <li>Item 1</li>
</ul>

<table border="1">...</table>
